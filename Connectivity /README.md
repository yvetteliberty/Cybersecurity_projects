# Packet Tracer  - Web Server Connectivity Analysis

 # 🔐Project Summary
This project demonstrates basic network  connectivity and web server access  using Cisco Packet Tracer .
It focuses on how a client communicates with a web server using IP addressing , ICMP, and HTTP protocols core concepts in networking and cybersecurity.

# 🎯 Learning Objectives
Verify network connectivity using ICMP (ping)
Understand IP-based communication across a network
Observe packet delivery and packet loss behavior
Access a web server using HTTP
Strengthen foundational networking and security concepts

# 🧰 Technologies & Tools
 - Cisco Packet Tracer
 - TCP/IP
 - ICMP
 - HTTP
 - Client–Server Model

   # 🧪 Lab Tasks & Observations

1️⃣ Connectivity Verification (ICMP)
Used ping to test reachability of the web server (172.33.100.50)
Successful replies confirmed network-layer connectivity
Initial packet loss observed due to ARP resolution

2️⃣ Web Server Access (HTTP)
Accessed the web server using a browser and IP address
The default web page loaded successfully
No HTTP errors encountered
